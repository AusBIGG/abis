== Requirements

Requirements are the individual rules that data claiming conformance to the ABIS must abide by. 

Requirements in ABIS are grouped according to several Conformance Classes detailed in the section above.

=== Requirements Index

==== <<Domain Model, Domain Model Conformance Class>>
* TERN Ontology
** All TERN Ontology Requirements as specified in model documentation
** See the https://linkeddata.tern.org.au/information-models/tern-ontology/conceptual-information-model[TERN Ontology Conceptual Model overview]
* GeoSPARQL
** All the Requirements relating to the GeoSPARQL ontology, not the GeoSAPRQL functions
** See the https://opengeospatial.github.io/ogc-geosparql/geosparql11/spec.html#_annex_a_abstract_test_suite_normative[GeoSAPRQL 1.1. Abstract Test Suite]
* Vocabulary Publication Profile
** See the https://w3id.org/profile/vocpub/spec[VocPub Specification's listing of Requirements]
* <<Domain Model Extended ABIS Requirements>>
** All of the Requirements specified in this section of this Specification
